## Commands

Clean up commands: as of docker 1.13 introduces clean-up commands.
To remove all unused containers, images, networks and volumes:

```
  docker system prune
```

Individually:

```
  docker container prune

  docker image prune

  docker network prune

  docker volume prune
```
